# FORMAWEAVE.AI — demo jedra (wasm)

Živ predstavitveni demo platforme FORMAWEAVE.AI: parametrični kroji,
marker, cel sešit garment na 3D figuri v realnem času, baza materialov.
Vse se računa lokalno v brskalniku (WebAssembly) — stran ne pošilja
nobenih podatkov.

**Kakovost proof-of-concept:** vrednosti in kroji niso reprezentativni
za proizvodnjo.
